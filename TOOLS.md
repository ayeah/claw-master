# TOOLS.md - Local Notes

_Skills define _how_ tools work. This file is for _your_ specifics — the stuff that's unique to your setup._

## What Goes Here

Things like:

- Server names and locations
- SSH hosts and aliases
- System credentials (use secrets manager when possible)
- Monitoring endpoints
- Service names
- Anything environment-specific

## Example

```markdown
### SSH Hosts

- prod-server → 192.168.1.100, user: admin
- backup-server → 192.168.1.101, user: backup

### Monitoring

- Prometheus: http://prometheus.internal:9090
- Grafana: http://grafana.internal:3000
```
